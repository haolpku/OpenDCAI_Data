# OpenDCAI · 训练数据集展示中心

一个可点击下钻的静态网页，用于展示 OpenDCAI / DataFlow 的训练数据样本。

- 12 个数据分类，每类 100+ 条真实样本
- 涵盖文本 SFT、数学推理 CoT、Text2SQL、代码生成、指令对话、医疗问答、函数调用、Agentic/多跳 RAG、预训练语料，以及多模态图像编辑（before/after 图像对）
- 纯静态站点（`index.html` + `data.js` + `assets/`），适配 GitHub Pages

## 本地查看

直接用浏览器打开 `index.html` 即可。

## 在线访问（GitHub Pages）

启用 Pages 后访问：`https://haolpku.github.io/OpenDCAI_Data/`
